#to Run

Backend (Spring Boot)
Open backend/ in your IDE (Eclipse/IntelliJ).
Set up MySQL:
Create a database named hotpotapl
Update DB credentials in src/main/resources/application.properties
Run the project as a Spring Boot app.
Frontend (Angular)
Open frontend/ in VS Code or terminal.
Run the app in bash:ng serve 
Open browser at: http://localhost:4200
